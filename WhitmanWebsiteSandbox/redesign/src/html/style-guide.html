Fonts

Source Sans Pro

Source Sans Pro is the primary font on the Whitman website. It can be seen in the body text, components and navigation elements throughout the site.

Helvetica / Arial

Helvetica is a secondary font on the Whitman website, used primarily for page titles. Since not all devices use Helvetica, a smililar font, Arial, or another sans-serif font may appear in its place.

*Please note that Heading 1 (<h1>) tags are automatically used for page titles by the content management system, and are only necessary once per page. They should not be manually added anywhere else.

It is important to remember that headings are for accessibility and organization and should always be used in the correct hierarchical order. They are not styling elements, and are not to be used based on visual preference.

For instance, <h3> headings must follow an <h2> heading or another <h3>. You cannot have <h3> or <h4> be the first heading of a page, nor can <h4> come before <h3> or <h2>.

Images

Images should be sized based on the area or component they occupy. They should be cropped to the appropriate size before being uploaded into Ingeniux. Having images that are too large drastically increase page load times and can create an unpleasant user experience. In most cases, the largest an image needs to be is the page content width: 882px wide. See below for the maximum image size for a given situation:

Maximum Image Sizes

Image Type Dimensions
Main header image 2400 x 1163px
Full-width body image 882px wide
Profile image 134 x 170px
Left/Right Column Component Image 294px wide

By default, images are displayed at the maximum size of the containing content area, regardless of actual size. Despite this, images should be no larger than the area or component they are used within.

Images should be chosen based on if they improve or assist the message of a webpage as necessary. Unless your page is a photo gallery, never use five images when just one will do — again, keep page load times and user experience in mind.
